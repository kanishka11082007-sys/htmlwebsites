# Basic HTML Website

A simple multi-page website built using **semantic HTML** as part of the roadmap.sh Frontend Developer roadmap.

## Features

- Semantic HTML5 structure
- Multi-page website
- Navigation between pages
- Projects page
- Articles page
- Contact page
- Responsive viewport meta tag
- SEO meta tags

## Pages

- Home
- Projects
- Articles
- Contact

## Technologies Used

- HTML5
